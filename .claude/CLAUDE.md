# CLAUDE.md

Guidance for Claude Code and other AI assistants working in this repo.

## Writing Markdown

Follow [docs/markdown-style.md](../docs/markdown-style.md) for all Markdown.

Most important rule: in **Korean** prose, never end a sentence with `).` — a
closing parenthesis immediately followed by a period. Rephrase (split the
sentence, fold the aside into the main clause, or use an em dash). This does
**not** apply to Markdown links/images such as `[text](url).`, whose trailing
period is correct and must stay.
