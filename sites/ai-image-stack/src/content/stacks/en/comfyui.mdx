## Overview

ComfyUI is a visual, node-based interface for running [[diffusion]] pipelines.
Instead of a fixed generation form, you wire nodes — model loader, sampler,
upscaler, post-processing — into a graph that is saved with every image, so any
result can be reproduced or remixed by loading it back.

It has become the de-facto workflow engine for local image generation: new
model families typically get ComfyUI support first, and the graphs it produces
double as shareable, runnable pipeline definitions.
