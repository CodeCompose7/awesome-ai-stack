## 무엇인가

**awesome-ai-stack**은 AI 시스템을 만들 때 실제로 쓰는 도구와 서비스를 모은
큐레이션 카탈로그입니다. 프레임워크, LLM 제공자, 벡터 스토어, 서빙, 관측 등을
다루며, 단순한 목록이 아닙니다. 항목마다 개요·메타데이터(언어·라이선스·가격)와
링크를 담은 상세 페이지가 있고, 대부분은 **바로 실행 가능한 코드 샘플**을 실어
"이게 뭐지"에서 "보여줘"까지 한 번의 스크롤로 이어집니다.

카탈로그 옆에는 여러 도구를 엮어 동작하는 스택을 구성하는 **개념**, 장문의 비교와
노트인 **글**, **프레젠테이션 슬라이드**, 그리고 반복되는 용어를 `[[위키링크]]`로
엮는 **용어집**이 있습니다. 모든 콘텐츠는 이중 언어로 제공되며, 영어는 루트에서,
한국어는 `/ko/` 아래에서 서빙됩니다.

## 어떻게 만들어졌나

이 사이트는 [Astro](https://astro.build)로 만든 정적 빌드이며 GitHub Pages로
배포됩니다. 모든 콘텐츠는 이 저장소 안에 MDX 파일과 작은 데이터 모듈로 있고,
이것들을 사이트로 만들어 내는 엔진 — 라우트·컴포넌트·스타일·마크다운 파이프라인
전부 — 은 별도의 Astro 테마
[stack-site-builder](https://github.com/CodeComposeStudio/stack-site-builder)이며
npm으로 설치합니다. 이렇게 나눈 덕분에 이 저장소에는 이 사이트를 이 사이트이게
하는 것 — 콘텐츠, 분류, 약간의 설정 — 만 남습니다.

## 기여하기

도구 하나를 추가하는 일은 로케일당 MDX 파일 하나면 됩니다. 카탈로그·둘러보기
페이지·백링크는 그 프런트매터에서 자동으로 따라옵니다. 도구를 추가하거나 샘플을
개선하거나 메타데이터를 고치는 기여를
[GitHub](https://github.com/CodeComposeStudio/awesome-ai-stack)에서 환영합니다.
