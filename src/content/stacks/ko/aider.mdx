## 개요

Aider는 "터미널에서 하는 AI 페어 프로그래밍"입니다. 변경을 대화로 설명하면 파일을 직접 편집합니다.  
git 친화적이라 모든 변경이 적절한 메시지와 함께 자동 커밋되므로, 히스토리가 깔끔하게 유지되고 어떤 편집이든 되돌리기 쉽습니다.

```mermaid
flowchart LR
  you[you] --> aider[Aider]
  aider --> model[model]
  model --> diff[edit + git commit]
```

**코드 샘플** 탭에는 대화형 세션과 스크립트형 단발 편집 예시가 있습니다 —
선택기에서 골라 비교해 보세요.

## 언제 쓰면 좋은가

기존 저장소를 빠르게 대화형으로 편집하면서 절제되고 되돌릴 수 있는 git 히스토리를
원할 때 — 그리고 LiteLLM으로 어떤 모델이든 — Aider를 쓰세요.
