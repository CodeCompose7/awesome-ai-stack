      model = SentenceTransformer("BAAI/bge-m3")
      docs = ["cats are great pets", "the stock market fell", "kittens love to play"]
      doc_emb = model.encode(docs)

      query = model.encode("tell me about cats")
      scores = model.similarity(query, doc_emb)[0]  # cosine similarity
      print(docs[int(scores.argmax())])  # -> a cat-related sentence

## 개요

Sentence Transformers(SBERT)는 다국어 `bge-m3` 같은 임베딩 모델을 내 컴퓨터에서 직접
실행해, API 호출 없이 텍스트를 벡터로 바꿉니다.  
RAG 파이프라인의 로컬·제공자 독립적인 절반입니다 — 임베딩은 여기서 만들고, 저장·검색은
벡터 스토어에서 합니다.

```mermaid
flowchart LR
  text[text] --> st[SentenceTransformer]
  st --> vec[vector]
  vec --> store[(vector store)]
```

**코드 샘플** 탭에는 텍스트 인코딩과 내장 유사도 검색 예시가 있습니다 — 선택기에서
골라 비교해 보세요.
