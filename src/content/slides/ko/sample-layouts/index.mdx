## 표

| 레이아웃 | 문법 | 비고 |
| --- | --- | --- |
| 표지 | `class="cover"` | 큰 제목, 중앙 |
| 배경 이미지 | `bg={img}` | 풀블리드 |
| 컬럼 | `:::cols` | `---`로 구분 |
| 이미지 분할 | `aas-split` | `r-30-70` 등 비율 |
| 이미지 정렬/채움 | `img-top`·`fill` | 세로 위치·cover |
| 부제 | `::sub[…]` | 제목 아래 작게 |
| 목차 제외 | `toc={false}` | 제목 숨김 |

표는 표준 마크다운 문법을 그대로 씁니다.

</Slide>

## 코드

코드 블록은 사이트와 동일한 Shiki 하이라이팅이 적용됩니다.

```ts
async function goToSlide(deck: HTMLElement, i: number) {
  const target = deck.querySelectorAll('.aas-slide')[i];
  deck.scrollLeft = target.offsetLeft; // 스냅 지점으로 이동
  return i;
}
```

## 이미지 세로 정렬
::sub[`img-top` · `img-bottom` — 이미지가 글보다 짧을 때 세로 위치]

<Image src={media} alt="AI 강의 배경" />
<div>

`img-top`이면 이미지가 위에, `img-bottom`이면 아래에 붙습니다 (기본은 가운데).

글이 길어 이미지보다 세로로 클 때, 이미지의 세로 위치를 정할 수 있습니다.

- 첫 번째 요점
- 두 번째 요점
- 세 번째 요점
- 네 번째 요점

</div>
