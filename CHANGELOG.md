# 변경 이력

이 프로젝트의 모든 주요 변경사항은 이 파일에 기록됩니다.

형식은 [Keep a Changelog](https://keepachangelog.com/ko/1.0.0/)를 따르며,
이 프로젝트는 [Semantic Versioning](https://semver.org/lang/ko/)을 준수합니다.

이 저장소는 AI 에이전트 도구·스택을 한눈에 비교하는 큐레이션 카탈로그 사이트
(awesome-agent-stack)입니다. Astro로 만든 정적 사이트이며 도구 카탈로그, 블로그,
실행 가능한 예제(`samples/`)를 한국어·영어로 제공하고 GitHub Pages로 배포됩니다.
1.0.0은 카탈로그·상세 페이지·예제·다국어를 갖춘 첫 정식 릴리스를 뜻합니다.

## 1.8.0 - 2026-06-30

개념·글에도 스택처럼 **카테고리 체계**를 도입하고, 개념·글의 **읽기 레이아웃을
스택과 통일**했습니다. 모델에 도구를 붙여 만드는 **실행 가능한 튜토리얼 두 편**(웹
검색·웹 스크래핑)을 새로 더했고, 라우트를 단수형으로 정리하면서 정렬·페이징과 모바일
사용성을 함께 손봤습니다.

### 추가됨 1.8.0

- **개념·글 카테고리/하위 카테고리** — 스택처럼 개념·글에도 전용 분류 체계를 도입.
  상세에는 브레드크럼, 분류별 탐색 페이지, 분류로 묶인 인덱스를 제공하고, 분류가 없는
  항목을 위한 "미분류" 폴백 분류를 둠
- **정렬 토글(이름순·최신순)** — 개념·글 인덱스와 분류 페이지에 정렬 토글을 추가. 개념은
  큐레이션 "기본" 순서도 선택지로 유지하고, 선택은 `?sort=`로 URL에 반영
- **페이지네이션(`?page=`)** — 분류 페이지의 평면 목록을 한 페이지 12개로 나눠 표시.
  12개 이하면 페이저는 숨기고, 정렬과 맞물려 동작
- **실행 가능한 튜토리얼 두 편** — Tavily 웹 검색 에이전트(오늘의 환율 묻기)와 Firecrawl
  웹 스크래핑 에이전트(문서를 마크다운으로)를 `samples/`와 글로 추가. 각 글에 LangGraph
  코드 워크스루와 실제 실행 결과, 히어로 이미지를 싣고, 임베드된 샘플의 관련 도구·파일을
  사이드 레일에 연결
- **"도구(Tools)" 개념** — 모델이 외부 세계와 상호작용하는 함수(함수 호출·MCP)를 다루는
  새 개념을 추가하고, 하네스 엔지니어링과 서로 링크
- **언어 공유 영어 헤딩 앵커** — 헤딩 id를 본문 텍스트가 아니라 영어로 고정해 한국어·영어가
  같은 앵커를 공유하고, 내용이 바뀌어도 외부 링크가 유지되게 함. 깊이에 따라 `#`/`##`/`###`로
  표시하고 h4까지 앵커를 부여
- **필터·정렬 상태 이어가기** — 카테고리 이동은 물론 상단 로고·탐색 링크에서도 가격·언어·
  라이선스·정렬 상태를 함께 전달
- **모바일 "맨 위로" 버튼** — 목차 레일이 없는 좁은 화면에서 우측 하단에 떠 있는 맨 위로
  버튼을 추가

### 변경됨 1.8.0

- **읽기 레이아웃을 스택과 통일** — 개념·글 상세를 1200px 폭과 본문 옆 in-flow 스티키 목차
  레일(왼쪽)로 재구성하고, 스택 개요에도 같은 읽기 타이포그래피(고딕 헤더 + 세리프 본문)를
  적용. 상단 메뉴 폭도 1200px로 맞춤
- **단수형 경로** — 라우트를 `/stacks/`·`/blog/`에서 `/stack/`·`/concept/`·`/article/`로
  단수화
- **카드 표시 정리** — 개념 카드를 글 카드와 맞춰 메타를 위로 올리고 알약형 태그로 통일,
  이후 글 카드는 "관련 개념"처럼 컴팩트 가로형으로 바꿔 목록 높이를 절약
- **관련 섹션 재배치** — 개념·글 상세에서 관련 개념을 위로, 관련 도구·관련 글을 맨 아래로 이동
- **파스텔 다이어그램** — 본문 Mermaid 다이어그램에 둥근 파스텔 테마와 하네스 영역 틴트,
  한국어 라벨을 적용
- **부드러운 화살표 아이콘** — 섹션 헤더의 ↗를 둥근 라인캡 SVG 화살표로 교체(개념·글·스택 공통)
- **읽기 표·앵커 여백 다듬기** — 본문 표에 둥근 모서리를 주고, `#categories`로 점프할 때
  검색바 위에 여백을 둠

### 수정됨 1.8.0

- 필터를 단 채로 들어와도 목록이 잠깐 걸러지지 않은 채 깜빡이지 않도록 처리하고, 패싯 옵션을
  전체 카탈로그에서 만들어 어느 페이지에서든 복원되게 함
- 탭 바와 샘플 뷰어 크롬은 읽기용 세리프가 아닌 고딕을 유지하도록 수정
- 모바일 "맨 위로" 버튼이 목차 레일 버튼과 셀렉터가 겹쳐 표시되지 않던 문제를 해결
- 도구 개념의 히어로 이미지, Mermaid 예약어 노드 id, 굵게 처리한 CJK가 깨지던 문제 등 콘텐츠 수정

## 1.7.0 - 2026-06-28

여러 도구를 하나의 패턴으로 묶어 설명하는 **개념(Concepts)** 콘텐츠 타입을 신설하고,
도구마다 **만든 곳** 정보와 글의 **문서 버전·업데이트 날짜**를 더했습니다. 도구 상세
헤더를 목록 카드와 같은 카드형 히어로로 다듬고, 표준 카드에도 갤러리 톤을 입혔습니다.

### 추가됨 1.7.0

- **개념(Concepts) 콘텐츠 타입** — 여러 도구를 엮어 하나의 패턴을 설명하는 새 콘텐츠
  타입. 하네스 엔지니어링을 파일럿으로, 도구를 역할별로 묶어 본문에 인라인 카드로
  보여주고 관련 글·관련 개념을 역링크로 연결했습니다. 도구 상세에는 "관련 개념"을
  카드로 표시
- **만든 곳(Vendor)** — 도구를 만들고 유지하는 주체 정보를 추가하고, 만든 곳별 모아보기
  페이지(`/vendors/<slug>`)를 신설
- **문서 버전·업데이트** — 개념·블로그·스택 글에 "문서버전"과 "업데이트 날짜"를 도입해
  도구 자체 소프트웨어 버전과 구분하고, 날짜는 언어별 형식으로 표시
- **개념·블로그 메인 이미지** — 두 콘텐츠의 상세와 목록 카드에 히어로 이미지를 더하고
  `astro:assets`로 WebP 최적화
- **오른쪽 목차 레일** — 개념·블로그 상세에 오른쪽 목차(TOC)와 "맨 위로" 버튼을 추가
- **관련 항목 접기** — 관련 도구·관련 개념이 6개를 넘으면 "+N개 더 보기" 토글로 접어
  본문이 아래로 밀리지 않게 처리
- **샘플 README 접기 + 예제 날짜** — 구현 샘플의 README를 접을 수 있게 하고, 각 예제의
  작성 날짜를 카드와 드롭다운에 언어별 형식으로 표시

### 변경됨 1.7.0

- **도구 상세 헤더를 카드형 히어로로** — 목록 카드의 디자인 언어를 상세 페이지에 적용해,
  테두리·둥근 모서리에 로고를 흐린 앰비언트 배경으로 깔고 별점과 컬러 가격 칩을 오른쪽에 배치
- **표준 카드에 갤러리 톤 적용** — 표준 보기 카드에도 블러 배경·둥근 모서리(1.5rem)·테두리
  없는 칩을 입히고, 라이트 테마 경계선을 #f4f4f4로 통일
- **관련 개념 카드** — 도구 상세의 관련 개념을 개념 메인 이미지가 왼쪽을 꽉 채우는 카드로 표시
- **명사로 끝나는 조각의 마침표 정리** — 한글은 명사로 끝나는 설명·목록·가격표 노트의 끝
  마침표를 제거하고 종결문은 유지, 영어는 명사구 조각만 제거. 규칙을 마크다운 스타일 가이드에 문서화

## 1.6.0 - 2026-06-27

목록을 더 조밀하게 한눈에 볼 수 있는 갤러리 보기를 더하고, 직전에 추가한 도구들에 실제
로고를 입혔습니다.

### 추가됨 1.6.0

- **갤러리 보기** — 목록을 조밀한 미니 카드로 보여주는 새 보기. 로고를 확대·블러한 배경
  위에 로고와 이름을 얹고, 별점·오픈소스 칩과 짧은 설명만 표시합니다. 배경은 아주 느리게
  움직이며, 접근성을 위해 `prefers-reduced-motion`에서는 정지합니다. 보기 선택은
  localStorage에 저장되고 기본값은 갤러리이며, 필터 바의 토글로 표준 카드와 전환합니다
- **신규 도구 로고** — 직전에 추가한 11종 중 10종에 실제 로고를 적용했습니다. Tavily는
  단색 마스크로 라이트·다크 테마에 모두 대응하고, 로고가 없는 NeMo Guardrails만
  모노그램으로 표시됩니다

## 1.5.0 - 2026-06-27

카탈로그를 11종 늘려 56개로 키우고 새 영역 3개(도구·연동, 가드레일, 음성)를 신설했으며,
도구별 가격을 가격 탭으로 정리하고(플랜별 표·출처·확인 기준일·신선도 배지) 카테고리
페이지 검색·필터와 URL 동기화를 더했습니다.

### 추가됨 1.5.0

- **새 스택 11종 + 새 카테고리 3종** — 카탈로그가 45개에서 56개로
  - 에이전트 도구·연동(신설): 웹 검색·수집(Tavily·Firecrawl·Exa), 브라우저·컴퓨터 사용
    (Browser Use·Stagehand), 코드 샌드박스(E2B), 도구 연동(Composio)
  - 가드레일(신설): Guardrails AI·NeMo Guardrails
  - 음성 에이전트(신설): Pipecat·LiveKit Agents
- **구현 샘플 2종** — `guardrails-ai_1`은 LLM 출력을 Pydantic 스키마로 검증하고,
  `nemo-guardrails_1`은 Colang 레일로 응답을 제어합니다. 둘 다 Docker로 실행하며 동작 검증
- **가격 탭** — 도구 상세에 가격 탭을 신설. 짧은 안내문과 함께 플랜·가격·비고를 표로
  보여주고, 출처 링크와 확인 기준일을 표기. 유료·클라우드 도구 22종에 플랜별 표를 넣고,
  API 제공자는 모델별 입력/출력 요율로 정리
- **가격 신선도 배지** — 확인 기준일이 6개월·1년을 넘으면 "가격 미확인" 배지를 표시.
  방문 시점의 현재 날짜를 기준으로 계산해, 정적 빌드가 오래돼도 정확하게 동작
- **카테고리·세부 카테고리 검색·필터** — 홈에만 있던 검색·필터·정렬 바를 카테고리
  페이지에도 추가. 가격·라이선스·언어 facet은 해당 페이지에 보이는 도구로 한정
- **필터 상태 URL 동기화** — 검색어·가격·라이선스·언어·최근 업데이트·정렬을 쿼리스트링에
  담아 반영하고 로드 시 복원. 필터된 화면을 링크로 공유·재현 가능

### 변경됨 1.5.0

- **헤더 내비게이션 아이콘 전용** — 모든 화면 폭에서 둘러보기·글·GitHub를 아이콘으로 표시
- **Agno 무료 사용 반영** — AgentOS 클라우드에 Free 티어가 있어 가격 정보를 정정
- **검색·필터 바를 공용 컴포넌트로 추출** — 홈과 카테고리 페이지가 같은 FilterControls를 공유
